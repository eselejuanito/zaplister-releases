# ZapLister

**Automate your Facebook Marketplace listings — post items, vehicles, and houses on autopilot.**

ZapLister is a desktop app that saves you hours by automatically publishing and managing your Facebook Marketplace listings. Set it up once, let it run.

🌐 **Website:** [zaplister.com](https://zaplister.com)

---

## Download

Go to the [Releases](https://github.com/eselejuanito/zaplister-releases/releases) page and download the installer for your platform:

| Platform | File |
|----------|------|
| Windows | `ZapLister-x.x.x-Windows.exe` |
| macOS (Apple Silicon) | `ZapLister-x.x.x-Mac-arm64.dmg` |
| macOS (Intel) | `ZapLister-x.x.x-Mac-x64.dmg` |
| Linux | `ZapLister-x.x.x-Linux.AppImage` |

---

## Features

- Post items, vehicles, and houses to Facebook Marketplace automatically
- Multi-account support
- Schedule and pipeline runs
- Supports English, Spanish, Portuguese, French, German, and Italian
- Auto-updates when a new version is released

---

## Requirements

- Google Chrome installed on your machine
- A valid ZapLister license — get one at [zaplister.com](https://zaplister.com)
